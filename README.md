**FreeJobSearcher** is a powerful, automated tool designed to streamline the job hunting process. By aggregating listings from multiple platforms and utilizing smart filtering, it helps developers and professionals find their next career move without the manual grind.

---

## Key Features

- **Profile Creation**
- **Resume Upload**
- **Mark others Profile favorite**
- **Upload Video:** for your Introduction or showcasing your soft skills like communication.
- **Multiple Job Categories:** can add single profile to multiple different job categories.

## Getting Started

### Prerequisites
Ensure you have the following installed:
- [Node.js](https://nodejs.org/en/download)
- [NPM](https://nodejs.org/en/download)

### Installation
1. **Clone the repository:**
   ```bash
   $ git clone https://github.com/the-loan-wolf/freejobsearcher-v3.git
   $ cd freejobsearcher-v3
   $ npm install
   $ npm run dev
   ```
